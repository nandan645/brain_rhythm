How to access the database?

The data that are available already on this website, the visitors could download it under the Creative Commons License.

Users can access the Brain Database directly from the navigation bar. Select Brain from the menu to open a dropdown list of available brain data categories, and then choose the dataset of interest. Each selection will direct the user to a dedicated page corresponding to the chosen data type.
